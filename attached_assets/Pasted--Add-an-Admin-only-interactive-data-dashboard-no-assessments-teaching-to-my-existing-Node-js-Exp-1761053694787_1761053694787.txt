
Add an Admin-only, interactive data dashboard (no assessments/teaching) to my existing Node.js (Express) + Postgres app.

Drilldown table (country → age_band → sector → job_nature) with expand/collapse and counts.

A Refresh button to re-fetch from the server and re-render.

A minimal /api/admin/ router:

Endpoint for small counters (e.g., total users, top country).

Endpoint returning aggregated counts for the 4-level drilldown (flat array is fine; build tree client-side).
